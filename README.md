# Data-Processing

Repositorio dedicado al **preprocesamiento y transformación de datos**, etapa fundamental previa al entrenamiento de modelos de machine learning y deep learning.

## 📂 Contenido

- **`Automatizacion datos JSON del event data.ipynb`** → Procesamiento automatizado de datos en formato JSON.
- **`Eventing data y los archivos JSON.ipynb`** → Manipulación y limpieza de datos de eventos a partir de archivos JSON.
- **`Limpieza y transformación de datos.ipynb`** → Técnicas generales de limpieza (valores nulos, outliers, normalización, etc.).
- **`SQL_To_Optimize_Queries.ipynb`** → Optimización de consultas SQL para extracción eficiente de datos.
- **`etl_nba.py`** → Script ETL (Extract, Transform, Load) para datos de la NBA (ejemplo práctico).

## ⚙️ Requisitos

- Python 3.7 o superior
- Jupyter Notebook / JupyterLab
- Librerías principales: `pandas`, `numpy`, `sqlalchemy`, `json`
